<h3 align="center"><a href="https://npanuhin.github.io/gfm-highlight-compare">Open online tool</a></h3>

Suppose you want to have a Markdown code block with some non-source code text, let's say a URL.

A standard code block can look a bit dull:

```
https://bing.npanuhin.me/{country}-{language}.json
```

But with the right language hint, you can make it stand out:

```d2lang
https://bing.npanuhin.me/{country}-{language}.json
```

Much better! This version uses the D2 language (`d2lang`) to add visual emphasis.

Here are some more examples:

```crystal
https://bing.npanuhin.me/{country}-{language}.json
```
